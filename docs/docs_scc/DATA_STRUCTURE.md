# SCC Data Structure

This document describes the main data structures used by Sportscar Control.

## Race object

Races are normally inserted or updated using:

```js
replaceRace({
  id: 'wec-fuji-2025',
  season: 2025,
  round: 7,
  series: 'wec',
  event: '6 Hours of Fuji',
  circuit: 'Fuji Speedway',
  country: 'Japan',
  date: '2025-09-28',
  entries: [],
  performance: {}
});
```

## Required race fields

- `id`
- `season`
- `round`
- `series`
- `event`
- `circuit`
- `country`
- `date`
- `entries`

## Optional race fields

- `scheduledDuration`
- `officialDuration`
- `scheduledLaps`
- `officialLaps`
- `sources`
- `completeness`
- `raceControl`

## Entry object

Each car entry should contain:

```js
{
  pos: 1,
  class: 'HYP',
  no: '51',
  constructor: 'ferrari',
  model: 'Ferrari 499P',
  team: 'Ferrari AF Corse',
  gap: '6:00:00.000',
  laps: 213,
  drivers: [
    ['Antonio Giovinazzi', 'Italy'],
    ['James Calado', 'United Kingdom'],
    ['Alessandro Pier Guidi', 'Italy']
  ],
  status: 'FIN'
}
```

## Car numbers

Car numbers must always be strings.

Correct:

```js
no: '007'
```

Incorrect:

```js
no: 7
```

## Driver list

Drivers are stored as pairs:

```js
['Driver Name', 'Country']
```

## Performance object

The performance object can contain:

```js
performance: {
  fastestLaps: [],
  poles: []
}
```

## Fastest lap item

```js
{
  type: 'FL',
  class: 'HYP',
  no: '50',
  team: 'Ferrari AF Corse',
  driver: 'Antonio Fuoco',
  lap: 89,
  time: '1:32.066'
}
```

## Pole position item

```js
{
  type: 'PP',
  class: 'LMGT3',
  no: '10',
  team: 'Garage 59',
  driver: 'Marvin Kirchhöfer',
  time: '1:41.181'
}
```

## Automatically calculated values

The app should calculate:

- Winner label
- Int
- Winning Margin
- Class Winners

These should not be hard-coded into race data.

## Race Control

Race Control is manual and optional.

It should not be generated by the PDF importer unless explicitly requested.

## Guiding principle

Race data should describe source results.

Shared app logic should calculate presentation values.
