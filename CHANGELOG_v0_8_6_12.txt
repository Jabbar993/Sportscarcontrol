SCC v0.8.6.12 - ACO/WEC importer pass

Base:
- Built from v0.8.6.11 / v0.8.6.5 database UI line.

Importer changes:
- Entry List is intentionally not part of the workflow.
- Results preview is rebuilt as a readable table: car, class, team, model and drivers stacked vertically with flags.
- Results import dedupes by car number before saving/importing. This is aimed at duplicate driver rows such as Juncadella showing up twice from the same car/session.
- Driver initials from Al Kamel PDFs are resolved against known SCC driver names when possible (example: D. JUNCADELLA -> Dani/Daniel Juncadella depending on database match).
- Performance import now treats ACO Qualifying / Hyperpole / Starting Grid PDF text as PP source.
- Race Classification PDF text remains usable as FL source.
- Save to database now writes to RACES and browser localStorage, then updates selectors/tree and shows a real success message with car/PP/FL counts.

Limits:
- Browser PDF text extraction still depends on PDF.js being available. If PDF.js cannot load, paste extracted text or CSV.
- This is an importer parser/UI pass, not a full server-side Alkamel downloader.
