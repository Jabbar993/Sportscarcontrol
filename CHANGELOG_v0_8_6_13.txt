SCC v0.8.6.13 · ACO import session

Importer changes
- Added an ACO import session panel in the import sidebar.
- Results and Performance upload fields now accept multiple files.
- Performance upload supports several PDFs at once, e.g. Fastest Laps + Hyperpole HYP + Hyperpole LMGT3.
- The session panel confirms for each PDF whether text was read and how many result cars / PP rows / FL rows were detected.
- Results preview keeps drivers stacked with flags and merges duplicate result rows by car number + class.
- Performance import combines PP and FL rows from multiple PDFs and keeps best Overall + best per class.
- Entry List remains intentionally outside the import workflow.
- Save to database stores the race in localStorage and shows a status message with car / PP / FL counts.
